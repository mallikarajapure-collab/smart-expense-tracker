📁 Smart Expense Tracker (Python) — README
🪙 Project Description

This is a simple console-based Expense Tracker built using Python. It helps users manage their daily expenses by adding, viewing, and analyzing spending in different categories.

🎯 Features
1,Add new expenses with name, category, and amount
2.View total expenses
3.ew category-wise expense list
4.Simple menu-driven interface
💻 Technologies Used

*Python (Core Concepts)
*Lists and Tuples
*Loops and Conditional Statements
🚀 How to Run
Download the file expense_tracker.py
Open terminal or any Python IDE
Run the file:
python expense_tracker.py
